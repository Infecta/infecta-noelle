# infecta-noelle

Code was only meant for me, just sharing it if anyone wants to see my poor code quality or wants to get it up and running themselves

## Running this
- Pre-requisites
  - MongoDB
  - Access to a bancho.py server


1. Clone repo
2. `npm i`
3. edit `.env.example` and save as `.env`
4. `npm .`
5. Done
